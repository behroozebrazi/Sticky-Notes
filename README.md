# Sticky Notes App

A simple Sticky Notes web application built with HTML, CSS, and JavaScript.

## Features
- Create notes
- Edit notes
- Delete notes
- Auto-save using Local Storage
- Responsive design

## Technologies
- HTML
- CSS
- JavaScript
